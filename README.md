## 📝 Personnalisation

### Modifier les Informations Personnelles

Éditez `index.html` et modifiez :
- Nom, titre, description
- Email, téléphone, localisation
- Compétences et formations
- Projets et langues

### Changer les Couleurs

Dans `style.css`, modifiez les variables CSS (lignes 5-30) :

```css
:root {
    --primary-color: #0B67D0;  /* Votre couleur principale */
    --secondary-color: #00d4ff; /* Votre couleur secondaire */
    /* ... */
}
```

### Ajouter une Photo de Profil

1. Ajoutez votre photo dans le dossier (ex: `photo.jpg`)
2. Dans `style.css`, modifiez `.profile-photo` :

```css
.profile-photo {
    background-image: url('photo.jpg');
    background-size: cover;
    background-position: center;
}
```
